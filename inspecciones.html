Version 3.0

  • Version 3.0:
    • Añadido sistema de suministro de almacén.
  • Versión 2.1:
    • Añadida corrección nameplate introducido por el montador. Corrije automáticamente al nameplate correcto cuando se introduce el unit serial number o el module serial number. No permite introducir en el campo del nameplate ningún valor cuyo primer caracter sea un número (debe comenzar con letra -D, M, X, J,...).
  • Version 2.0:
    • Añadida la opción de exportar la lista de inspecciones finalizadas tanto a PDF como a Excel.
    • La tabla de inspecciones finalizadas no se refresca cada 60 segundos en el caso de que se esté consultando una fecha distinta a la actual. De este modo se dispone de más tiempo para realizar la consulta. NOTA: esta tabla vuelve automáticamente a la fecha actual a los 10 minutos.
    • Pequeños cambios cosméticos. Se agranda el recuadro que muestra la fecha en la tabla de inspecciones finalizadas para mejor visibilidad.
    • Se añade icono "deshacer" junto al nombre del inspector que permite deshacer el inicio de una inspección en caso de hacerlo por error.
